# 🎨 Banana Toolbox - UI 设计文档

## 一、整体风格

**设计语言：** Material Design 3 (Material You)

**设计原则：**
- 简洁高效，减少视觉干扰
- 操作直观，降低学习成本
- 信息层次分明，重点突出
- 动效流畅自然，反馈及时

---

## 四、通用组件

### 4.1 文件列表项
- 图标（根据文件类型自动匹配）
- 文件名（超长省略）
- 大小 / 修改时间
- 多选复选框（多选模式）

### 4.2 操作对话框
- 确认对话框（删除/覆盖等危险操作）
- 进度对话框（复制/移动等耗时操作）
- 属性对话框（文件详细信息）

### 4.3 底部操作栏
- 多选模式时显示
- 包含常用操作按钮
- 支持自定义操作

---

## 五、动效规范

| 场景 | 动效 |
|------|------|
| 页面切换 | 滑入/滑出 300ms |
| 卡片点击 | 缩放反馈 100ms |
| 列表滚动 | 渐显渐隐 |
| 删除操作 | 滑出消失 |
| 加载状态 | 骨架屏 / 进度条 |
